# Updates Staging Contract

This directory is generated by `scripts/release-macos.sh` and published to GitHub Pages by `.github/workflows/release.yml`.

Expected outputs for each tag release:

- `appcast.xml`
- `GLMBar.zip`
- Optional `GLMBar.md` release notes

Enclosure URL policy:

- Every new appcast item must point to:
  - `https://github.com/uwseoul/glm-bar/releases/download/vX.Y.Z/GLMBar.zip`

Where `vX.Y.Z` is the Git tag used for the release run.

Do not commit private signing keys to this directory.
